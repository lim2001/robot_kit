# shell模块

## 功能说明：

shell 模块提供开发者常用人机交互调试指令入口，方便开发者调试测试开发

## 机器人实物：

|                      |          |                                      |                                         |

## 应用说明：

shell 模块提供通用型快捷指令调试测试交互入口和解析器。

### 编译

1、可以全编译 colcon build

2、单独编译本功能包 colcon build --packages-select shell

### 运行

ros2 run shell shell
