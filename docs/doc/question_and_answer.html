Q&A

问题1：奥比robot_kit 采用哪种推荐开发环境?

【答】robot_kit 开发包提供开发者二次开发完善的开发环境，提供支持docket和apt install模式

问题2：奥比robot_kit 怎样编译?

【答】robot_kit 提供支持docket和apt install环境，docket编译采用ros2编译指令
colcon build
colcon build --packages-select <编译模块>
